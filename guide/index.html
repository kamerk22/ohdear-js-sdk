# An introduction to the Oh Dear! Javascript SDK

The OhDear Javascript library provides convenient models and methods to access the OhDear API from applications written in JavaScript.

Before moving forword we suggest you to checkout OhDear REST API documentation. It'll help you get better understanding.

  • Read the getting started guide for the SDK.

The SDK currently supports the following services:

API Name Class Name Status
Sites OhDear.Site 🎉
Checks OhDear.Check 🎉
Broken Links OhDear.BrokenLink 🎉
Mixed Content OhDear.MixedContent 🎉
Uptime OhDear.Uptime 🎉
Downtime OhDear.Downtime 🎉
Certificate Health OhDear.CertificateHealth 🎉
User Info OhDear.UserInfo 🎉
Status Pages OhDear.StatusPage 🎉
Status Page Update OhDear.StatusPageUpdate 🎉
